# 第一部分：项目介绍

## 1.1 这是什么？

**MarkdownPasteAddin** 是一套完整的「网页内容 → Word 文档」转换工具链。

从 DeepSeek、ChatGPT、网页中复制的**表格、Mermaid 流程图、LaTeX 数学公式、代码块**等内容，一键转为格式规范、可直接使用、甚至可以直接提交的 Microsoft Word 文档。

## 1.2 解决的核心痛点

```
用户在 AI 平台得到回答 → 复制粘贴到 Word → 格式全乱 → 手动重做表格/公式/流程图
                                                                    ↓
MarkdownPasteAddin:            一键粘贴 → 自动生成格式完美的 Word 文档
```

## 1.4 功能清单

| 类别 | 功能 | 输入 | 输出 |

| **表格** | Markdown/Html 表格 | `\| A \| B \|` | Word 原生表格(蓝底表头) |

| **目录** | TOC 域代码 | 选项开启 | Word 目录(更新域可见) |
| **图表编号** | 自动递增 | 图片/表格/流程图 | 图1/表1 自动标注 |

# 第三部分：技术亮点

## 3.1 LaTeX → OMML 数学公式引擎

自主实现的递归下降解析器，将 LaTeX 公式直接转为 Word 原生 OMML XML 对象：

```
输入: \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
  ↓ 词法分析 → 递归下降解析 → AST
  ↓ AST → OMML XML 生成
输出: Word 原生公式对象 (可编辑、可排版)
```

支持: 分式、根号、上下标、希腊字母、矩阵、积分求和、重音符号、分隔符

## 3.2 三套专业报告格式标准

```
政府公文 (GB/T 9704):     学术论文:              商务报告:
  仿宋 16pt                宋体 12pt              Calibri 11pt
  黑体/楷体 标题           黑体 标题               Arial/黑体 标题
  上3.7 下3.5 页边距       左3.17 右3.17           2.54cm 均匀
  章/节/条 编号            1/1.1/1.1.1 编号        现代简洁风格
```

## 3.3 跨浏览器扩展架构

同一份扩展代码兼容 Chrome 88+ 和 Edge 102+，通过 browser-polyfill.js 标准化 API：

```
Extension Code → browser-polyfill.js → chrome.* (Chrome/Edge)
                                     → browser.* (Firefox)
```

## 3.4 内容捕获的通用性

不依赖特定平台 DOM 结构，三种抓取策略：

1. **AI 对话识别**: 扫描 `[class*=assistant]`、`[data-message-author-role]` 等通用属性
2. **结构化内容检测**: 自动发现页面中的代码块、表格、Mermaid 图
3. **选区导出**: 用户手动框选 → 右键导出

3. 填写商店信息:

| 字段 | 内容 |
|------|------|
| Name | MarkdownPasteAddin - Smart Paste to Word |

| Description | 从 STORE_LISTING.md 复制英文描述 |
| Category | Productivity |

4. 上传素材:

| 素材 | 路径 |
|------|------|

### Step 3: 审核 (通常 1-3 天)

常见被拒原因及对策:

| 问题 | 解决方案 |
|------|----------|
| 权限过宽 | 说明 host_permissions 仅用于 localhost 桥接服务 |
| 功能描述不匹配 | 确保截图与功能描述一致 |
| 缺少隐私政策 | 在描述中明确 "No data collection" |
| 关键词堆砌 | 标题和描述自然写，不堆砌关键词 |

### Step 4: 发布后

- 审核通过后扩展自动上架
- 可以在 Developer Dashboard 查看安装量、评分
- 更新: 修改代码 → 在 manifest.json 中递增 version → 重新打包上传
